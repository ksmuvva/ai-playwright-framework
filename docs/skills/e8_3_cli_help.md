# E8.3 - CLI Help System

**Skill:** `e8_3_cli_help`
**Version:** 1.0.0
**Author:** Claude Playwright Agent Team

## Overview

Provides comprehensive help system for CLI commands including usage examples and documentation.

## Capabilities

- Display command help
- Show usage examples
- Provide command documentation
- Generate man pages

## Usage

```bash
cpa --help
cpa init --help
cpa test --help
```
